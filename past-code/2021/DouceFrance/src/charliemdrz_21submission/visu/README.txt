Let's try the provided example: Visualization of a handmad Minecraft map with accessibility heatmap.

Write the following commands:

python pre_processing.py			(creates both map and heatmap as .png files in stock\examples_50x50\)

python map_visu.py example_50x500		(launch the Kivy visualization tool)





Note: If Kivy is not installed, please use the following command:

conda install kivy -c conda-forge
